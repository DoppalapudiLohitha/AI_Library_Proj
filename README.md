## вљ™пёЏ Installation

### 1. Clone the repository

```bash
git clone https://github.com/DoppalapudiLohitha/AI_Library_Proj.git
```

### 2. Open the project

```bash
cd AI_Library_Proj
```

### 3. Install the required libraries

```bash
pip install -r requirements.txt
```

Open **MySQL Workbench** and execute:

